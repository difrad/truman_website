About Truman

Truman is a research platform for testing theories of online behavior and communication. The platform allows participants in social psychology related research studies to interact with each other or with artificial “bot” users in a controlled environment. Developer could use the platform to create different interfaces to feed the different needs of research.

API Used in This Project

Truman platform is created for social media experiment. Currently, it does not need any of the included APIs or OAuth authentication methods of social media website. But API keys can be obtained for function/feature developing. Here is useful hints for generating API credentials

https://github.com/sahat/hackathon-starter#obtaining-api-keys
