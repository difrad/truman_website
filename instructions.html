Instructions

Installing

Truman is a research platform for testing theories of online behavior and communication. The platform allows participants in social psychology related research studies to interact with each other or with artificial “bot” users in a controlled environment. Researchers could modify the identity of the robots and customize the study environment. The following tutorial will display the process of uploading robot data into the Truman platform.

Download the data upload wizard from here

Homepage

After downloaded the platform, the user will see the home page of the uploading wizard. The wizard allow user create bots lists in two ways: Auto generate and Uploading customized spreadsheet. If user already created their own spreadsheet, they can click the “Import” button to access following steps. The wizard is also able to help user auto-generate the information in bots lists. The user can use the auto generate function by clicking the “Generate” button.

Import

If the user chooses to upload their own spreadsheet, the “Import” button will bring them to the import page. User could upload data by dragging and dropping a CSV, XLS, or XLSX file into the window. After uploaded, the spreadsheet will appear on the button of the page. User could edit every cells in the spreadsheet and download the spreadsheet into their own device.







Generate

User could also create a robot list without their own spreadsheet. The auto generate function of the Wizard is able to help the user generate a random bot lists. User need to input the number of bots they want. The Wizard will create an editable spreadsheet.
